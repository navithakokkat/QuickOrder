Usage Instructions
------------------
1. Install the template engine Pug by moving to the directory of this project in a terminal and using the command `npm install pug`. 
2. Run the server using the command `node server.js`. The url will be displayed. 
3. Copy and paste that url into a web browser.
4. To stop the server, in the terminal press Ctrl + C and hit enter.
